# FolderTree 
 This method prints out a folder tree of the path passed in.

 Format of the folder tree is:
 ```
- Root
-  Subdirectory 1:
   - file11
   - file12
   - Sub-sub-directory 11:
         - file111
         - file112
- Subdirectory 2:
    - file21
    - sub-sub-directory 21
    - sub-sub-directory 22    
        - sub-sub-sub-directory 221
            - file 2211
```
